## Features

- Client and employee management with ID validation
- Vehicle registration and status tracking
- Entry and exit recording with timestamps
- Monthly contracts linked to clients and vehicles
- Additional services: car wash, tire change, interior cleaning, light replacement
- Reports module with movement history, top clients, employee activity and longest stays
- Dark/light theme toggle
- Built-in mini game
- Data persistence via Java serialization

## Tech Stack

- JDatePicker
